КАК УСТАНОВИТЬ?

Windows:
  1. 1. Скачайте файл .exe
  2. 2. Запустите установщик
  3. 3. Следуйте инструкциям на экране
  4. 4. Готово — программа запущена
Linux (Debian / Ubuntu):
  1. 1. Скачайте .deb или .rpm
  2. 2. Откройте терминал
  3. 3. sudo dpkg -i склад.deb
  4. 4. Запустите из меню приложений
