## Описание

Разработка веб сайта по учету товаров, их приход и расход, составление отчета по взаимодействию с товарами за определенный промежуток времени (за день, за неделю, за месяць)

## Основные задачи

* Фронт:
  1) Интуитивно понятный интерфейс
  2) Сбалансированная цветовая палитра
  3) Дизайн сайта
* Бэк:
  1) Разделенная логика
  2) Запись комментов при необходимости

## Второстепенные задачи

* Фронт:
  1) Плавные анимации
  2) Правильная и хорошо построенная структура (использование base, include и прочее)
  3) У стилей и скриптов одна обязанность (то есть, все не написано в один файл)
  4) Понятное наименование файлов в соответствии с его ролью
  5) DRY (Don't Repeat Yourself)
  6) Использование TailwindСSS
* Бэк:
  1) Оптимизация процессов, быстрая загрузка
  2) Написание тестов
  3) DRY (Don't Repeat Yourself)
  4) Использование Redis для кэширования
